## üåê API Endpoints

| Method | Endpoint        | Description        |
|--------|------------------|--------------------|
| GET    | /api/get         | Get all todos      |
| POST   | /api/post        | Add a new todo     |
| PUT    | /api/update/:id  | Update todo        |
| DELETE | /api/delete/:id  | Delete todo        |
